## magpie
Rapid Prototyping of Computer Systems 2017

# Install node
https://nodejs.org/en/

# Install postgres
https://www.postgresql.org

# Create a database magpie in PostgreSQL
psql
create database magpie;

# Install dependencies
npm install

# To create table
node models/(filename)

# Start server
npm start

# Client side 
http://localhost:3000/
